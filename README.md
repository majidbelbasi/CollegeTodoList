مشخصات پروژه
نام پروژه: لیست کارهای روزانه

نام دانشجو: مجید بلباسی

تاریخ تحویل:1404/11/24

شرح پروژه
این یک برنامه ساده برای مدیریت کارهای روزانه است که با استفاده از HTML، CSS و jQuery ساخته شده. کاربر میتواند کارهای خود را به لیست اضافه کرده و بعد از انجام، آنها را حذف کند.

امکانات پروژه
افزودن کار جدید با کلیک دکمه "افزودن"

افزودن کار جدید با فشردن کلید Enter

حذف هر کار با کلیک روی دکمه "حذف"

انیمیشن هنگام اضافه و حذف شدن کارها

طراحی زیبا و کاربرپسند

آموزش کار با برنامه:

در کادر سفید، کار خود را بنویسید

روی دکمه "افزودن" کلیک کنید یا کلید Enter را بزنید

کار شما به لیست اضافه میشود

برای حذف هر کار، روی دکمه "حذف" کنار آن کلیک کنید

نتیجه گیری
پروژه با موفقیت انجام شده و تمام امکانات مورد نظر پیاده سازی شده است. کاربر میتواند به راحتی کارهای روزانه خود را مدیریت کند.
